# node.js
## node.js 기본세팅
1. 원하는 파일경로에 cmd실행
2. npm init -> 항목 모두 엔터
3. npm install express (express 라이브러리 설치)
4. npm run dev 서버 실행

## 알아둘점
> require - 모듈 불러올 때 사용.

### 통신규격
- HTTP, HTTPS
- SSH, SCP
- FTP
- TEI

### Http Method
- get -> get
- post -> new, login
- put or patch -> modify, update (멱등성)
- delete -> del..

### session & cookie
>session - 서버에 정보등을 저장하는 것(쿠키사용)  
cookie - 웹에서 이용된 환경설정 및 기타 정보를 사용자의 컴퓨터에 작은 파일로 저장한 것. 만능. 이거 삭제하면 세션도 같이 삭제.
