# node.js
## node.js 기본세팅
1. 원하는 파일경로에 cmd실행
2. npm init -> 항목 모두 엔터
3. npm install express (express 라이브러리 설치)

## 알아둘점
### 통신규격
- HTTP, HTTPS
- SSH, SCP
- FTP
- TEI
### Http Method
- get -> get
- post -> new, login
- put or patch -> modify, update (멱등성)
- delete -> del..
